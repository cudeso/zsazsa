Single sign-on
zsazsa identifies the current user from MISP's own session cookie, which only works when zsazsa is deployed as a subpath behind MISP (e.g. /zsazsa/) so the browser sends MISP's session cookie along with requests. When enabled, visitors without a valid MISP session are redirected to MISP's login page. When disabled, requests without a MISP session fall back to the admin@admin.test identity used throughout zsazsa.
MISP names its session cookie MISP-<instance uuid>, which is unique to each MISP install. You do not need to set this by hand: when you enable single sign-on and save, zsazsa detects it from the connected MISP server and stores it in MISP_SESSION_COOKIE_NAME.

            Reasoning models (the o-series, e.g. o4-mini) are supported, but they
            spend part of their token budget on internal reasoning before producing output. The per-feature
            token limits in this app are tuned for standard chat models, so a reasoning model may exhaust its
            budget and return empty or truncated output. Use them only where you know the limit is generous
            enough; otherwise prefer a gpt-4o-family model.

Data collection: tag strip prefixes

        

          

            When a tag starts with one of these prefixes, the prefix is stripped and only the value is shown as a badge.
            Tags starting with misp-galaxy: also receive a
             galaxy icon.
            One prefix per line. Include the trailing = where applicable.
          

          
          
Example: misp-galaxy:malpedia= turns misp-galaxy:malpedia=Emotet into Emotet.

        

      


      

        
Data collection: hidden tag prefixes

        

          

            Tags whose prefix matches one of these entries are hidden completely from the data-collection tag display
            and from the tag-cloud filter. They may still appear in the scraper metadata tooltip.
            One prefix per line.
          

          
          
Example: retention hides retention:99d from the event tag display.
