Predefined actions shown as checkboxes in the flash intel and VEA wizards. One action per line. Users can still add free-text actions on top of these.

Data collection: tag strip prefixes

When a tag starts with one of these prefixes, the prefix is stripped and only the value is shown as a badge. Tags starting with misp-galaxy: also receive a galaxy icon. One prefix per line. Include the trailing = where applicable.

Example: misp-galaxy:malpedia= turns misp-galaxy:malpedia=Emotet into Emotet.
Data collection: hidden tag prefixes

Tags whose prefix matches one of these entries are hidden completely from the data-collection tag display and from the tag-cloud filter. They may still appear in the scraper metadata tooltip. One prefix per line.

Example: retention hides retention:99d from the event tag display.
